Este ejercicio copia el contenido de un fichero llamado origen.txt a otro llamado destino.txt, utilizamos BufferedReader para leer el archivo original linea por linea y bufferedWriter para escribir cada línea en el nuevo archivo, añadiendo un salto de línea con newLine(). Ambos flujos se gestionan dentro de un try. cuando termian mostramos un mensaje diciendo que se copio correctamente, añadimos IOException.
